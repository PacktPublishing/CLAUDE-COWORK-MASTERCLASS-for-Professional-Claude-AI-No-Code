SETUP
-----
If you completed Lesson 11, you already have a PashunSports/ folder
on your machine with folder access granted to Cowork.
Copy the Suppliers/ folder contents into your existing PashunSports/Suppliers/.

If this is your first lesson using PashunSports/:
Copy the entire PashunSports/ folder to your computer, then grant
Cowork access to it when prompted during your first task.

THE DEMO TASK — Step 1 (assign and review the plan)
-----------------------------------------------------
Run this in Cowork (swap the path to match your machine):

    Organise my /PashunSports/Suppliers/ folder.
    Group files by supplier name.
    Rename each file to include the supplier name and date.
    Show me the plan before making any changes.

Cowork will respond with a numbered plan. Read it carefully before approving.

THE DEMO TASK — Step 3 (refine before approving)
-------------------------------------------------
After reviewing the plan, redirect it:

    Don't create subfolders. Keep everything in the same directory.
    Just rename the files using: SupplierName_Date_Description.
    Then proceed.

THE DEMO TASK — Step 4 (intervene mid-task)
--------------------------------------------
If you want to practise a mid-task intervention, type this while
Cowork is running:

    Stop. Skip the FabricFirst files — I need to check those manually.

KEY LESSON POINTS
-----------------
- Always include 'show me the plan first' in new or complex task briefs
- Read the plan before approving — check file counts, paths, and steps
- You can refine the plan before approving
- You can stop or redirect at any point during execution
- Typing 'Stop' immediately halts the task
