LESSON 11 — How To Grant Folder Access So Claude Can Read, Edit, And Create Files
==================================================================================

WHAT'S IN THIS FOLDER
----------------------
PashunSports/    A pre-built empty workspace — all subfolders ready,
                 no files inside them yet.

  Orders/        Will hold order files in later lessons.
  Products/      Will hold product specs in later lessons.
  Suppliers/     Will hold supplier docs in later lessons.
  Reports/       Cowork will write output here.
  Marketing/     Cowork will write output here.
  WorldCup2026/  Will hold World Cup launch materials.

SETUP
-----
Copy the PashunSports/ folder to your computer.
  Mac example:     /Users/yourname/Documents/PashunSports/
  Windows example: C:\Users\yourname\Documents\PashunSports\

This is the folder structure you will grant Cowork access to
in the lesson. Granting access to the root PashunSports/ folder
covers all subfolders automatically — you only do this once.

THE README DEMO TASK
---------------------
In this lesson, Cowork creates a README.txt from scratch to confirm
folder access is working. Run this in Cowork:

    Create a setup confirmation file in my /PashunSports/ folder.
    Create a file called 'README.txt' containing:
      - Today's date
      - The text: 'Pashun Sports Cowork workspace initialised.'
      - A list of the subfolders we'll use:
        Orders, Products, Suppliers, Reports, Marketing, WorldCup2026
    Save it directly in /PashunSports/

When Cowork asks which folder to access, select your PashunSports/
folder. If README.txt appears there — folder access is working.

WHAT TO GRANT AND WHAT NOT TO GRANT
-------------------------------------
Grant:        PashunSports/ and its subfolders
Don't grant:  Your home directory, Desktop, or Downloads
Don't grant:  Folders with passwords, banking details, or sensitive data
