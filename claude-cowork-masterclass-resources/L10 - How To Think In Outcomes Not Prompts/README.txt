LESSON 10 — How To Think In Outcomes, Not Prompts
==================================================

WHAT'S IN THIS FOLDER
----------------------
PashunSports/
  Orders/
    2026/         12 order files with deliberately inconsistent naming
                  — mixed date formats, vague names, duplicates.
                  This is the starting point for the organise demo.
  Products/
    WorldCup2026/ 5 product spec sheets, design notes, and a pricing
                  CSV — used in the World Cup launch brief demo.
  Reports/        Empty. Cowork saves output here.
  Marketing/      Empty. Cowork saves the launch brief here.

SETUP
-----
Copy the PashunSports/ folder to your computer.
Note the full path — you will reference it in your Cowork briefs.
  Mac example:     /Users/yourname/Documents/PashunSports/
  Windows example: C:\Users\yourname\Documents\PashunSports\

DEMO 1 — The Outcome Approach (organising the Orders folder)
-------------------------------------------------------------
Run this in Cowork (swap the path to match your machine):

    I want my /PashunSports/Orders/2026/ folder organised so I can
    find orders fast. Please:
    - Group files by month (Jan, Feb, Mar, etc.)
    - Rename each file to: YYYY-MM-DD_OrderID_CustomerName
    - Flag any files that don't follow this naming pattern
    - Create a summary CSV listing all files with their new names
      saved to /PashunSports/Reports/OrderIndex_Apr2026.csv
    Show me your plan before making any changes.

DEMO 2 — World Cup Launch Brief
---------------------------------
Run this in Cowork (swap the path to match your machine):

    I need a product brief document for our 2026 World Cup launch range.
    Source files: /PashunSports/Products/WorldCup2026/
    Output: /PashunSports/Marketing/WorldCupLaunch_Brief_Apr2026.docx
    Include:
      - List of all products with names, SKUs, prices, and target markets
      - Top 3 hero products (flag from the pricing CSV)
      - Suggested launch order by territory (UK, Brazil, Argentina)
    Tone: confident, direct, business-ready. Max 500 words.
    Show me the plan first. Don't create the file until I approve.
