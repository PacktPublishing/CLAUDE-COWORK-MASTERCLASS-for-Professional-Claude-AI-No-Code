LESSON 22 — How To Use Claude In Chrome To Browse And Research On Your Behalf
=============================================================================

WHAT'S IN THIS FOLDER
----------------------
PashunSports/
  Products/
    CurrentRange_2026.xlsx    The full current product range — 9 products with
                               SKU, name, territory, colourway, RRP, sizes,
                               stock level, and status (Live / Low Stock / In Review).
                               Used in Workflow 3 (trend comparison).

ABOUT THIS LESSON
-----------------
Workflows 1 and 2 are browser-based — no local input files needed:
  Workflow 1: Browse a competitor site -> Claude summarises products
  Workflow 2: Browse FabricFirst portal -> Claude extracts pricing table

Workflow 3 combines Chrome browsing with this local file:
  Step 1: Use Chrome to visit google.com/trends — search 'world cup 2026
          merchandise' and note trending topics and rising searches.
  Step 2: Read /PashunSports/Products/CurrentRange_2026.xlsx
  Step 3: Compare trending styles to the current range.
           - Which trends match products we already carry?
           - Which are gaps we could target?
  Step 4: Save a brief analysis to:
           /PashunSports/Marketing/TrendAnalysis_Apr2026.docx

SETUP
-----
Install Claude in Chrome first:
  Claude Desktop -> initials (bottom-left) -> Settings -> Connectors ->
  Claude in Chrome -> toggle on -> Download extension -> Add to Chrome -> Pin it.

Then navigate to any website and click the Claude icon in your toolbar.
