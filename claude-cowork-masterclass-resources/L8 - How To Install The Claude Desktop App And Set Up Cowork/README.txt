LESSON 8 — How To Install The Claude Desktop App And Set Up Cowork
===================================================================

WHAT'S IN THIS FOLDER
----------------------
Documents/    An empty folder used in the verification task demo.
              Cowork will create 'cowork_test.txt' here during the lesson.

BEFORE THE LESSON
-----------------
1. Make sure you are on Pro plan or above (Lesson 7).
2. Go to claude.ai/download and download Claude Desktop for your OS.
   Supported: macOS (Apple Silicon), Windows x64.
   NOT supported: Intel Mac, Windows ARM64.
3. Install and sign in with your Claude account.
4. Click the Cowork tab in the left sidebar.
5. Wait for Cowork to finish its first-launch download (~2GB, a few minutes).
6. The desktop app must stay open while Cowork runs.

THE VERIFICATION TASK
----------------------
Once Cowork is set up, type this into the Cowork task input:

    Create a text file in my Documents folder called
    'cowork_test.txt' containing today's date and the
    words 'Cowork is working correctly.'

When Cowork asks which folder to access, point it to the
Documents/ folder included in this package.

If cowork_test.txt appears in that folder — setup is complete.
