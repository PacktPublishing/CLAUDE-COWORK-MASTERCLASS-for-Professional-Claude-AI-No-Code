LESSON 16 — How To Organise Messy Folders Using Cowork
=======================================================

WHAT'S IN THIS FOLDER
----------------------
PashunSports/
  Downloads/    25 deliberately messy files:
                  8 spec sheets - inconsistent names and capitalisation
                  1 pricing CSV - confusing name
                  1 design notes file
                  4 campaign brief drafts (including superseded v1)
                  3 supplier pricing files
                  8 unclear files - Cowork should flag these for Review

Cowork will create Products/WorldCup2026/, Marketing/, Suppliers/,
Reports/, and Review/ when it reorganises the folder.

SETUP
-----
Copy the PashunSports/ folder to your computer.
Grant Cowork access to PashunSports/ when prompted.

DEMO STEP 1 — Ask Cowork to scan and propose
----------------------------------------------
    Help me organise my /PashunSports/Downloads/ folder.
    Scan the contents and propose a plan:
      - What categories/folders should I create?
      - How should files be sorted into them?
      - What naming convention should I apply?
      - Any files to flag for review or deletion?

    Show me the plan and wait for my approval before making any changes.

Expected proposal:
  /Products/WorldCup2026/  -> 8 spec sheets + pricing CSV + design notes
  /Marketing/              -> 4 campaign brief drafts
  /Suppliers/              -> 3 supplier pricing files
  /Review/                 -> 8 unclear files

Review and approve. Cowork reorganises everything.

DEMO STEP 2 — Build the lookbook from clean files
--------------------------------------------------
Once organised, follow up in the same task:

    Now read all files in /PashunSports/Products/WorldCup2026/
    and produce an HTML product range lookbook:
      - Show each tee as an SVG flat-lay in its correct colourway
      - Include SKU, RRP, target territory, and launch status
      - Group by territory: UK, Brazil, Argentina
    Save as: /PashunSports/Marketing/PashunSports_Lookbook.html
