LESSON 34 — How To Use Dispatch To Assign Desktop Tasks From Your Phone
=======================================================================

WHAT'S IN THIS FOLDER
----------------------
PashunSports/
  Orders/
    Daily/
      orders_daily_20260611.csv     Today's order export (World Cup opening day).
                                     6 orders across different production statuses.
                                     Cowork reads this for Scenario 1.

  Suppliers/
    Invoices/
      Wk18/
        KitPro_Invoice_KP-WK18-001.txt
        PrintForce_Invoice_PF-WK18-001.txt
        FabricFirst_Invoice_FF-WK18-001.txt

        Three supplier invoices for Week 18.
        Cowork reads these for Scenario 2 and generates the cost report.

SETUP
-----
Copy the PashunSports/ folder to your computer.
Set up Dispatch before running the demos (see slides for setup steps).

SCENARIO 1 — Morning Briefing (send from phone)
-------------------------------------------------
    Run the daily order briefing for Pashun Sports.
    Read today's order export from /PashunSports/Orders/Daily/
    Save the briefing to /PashunSports/Reports/Daily/ as usual.

SCENARIO 2 — Supplier Cost Report (send from phone)
-----------------------------------------------------
    Generate the Week 18 supplier cost report from the files in
    /PashunSports/Suppliers/Invoices/Wk18/
    Save to /PashunSports/Reports/SupplierCosts_Wk18.xlsx
    Message me when it's done.

NOTE
----
Both tasks are sent from your phone via the Claude mobile app (Dispatch).
Your desktop must be on with Claude Desktop open for them to execute.
